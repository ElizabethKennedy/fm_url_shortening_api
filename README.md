I built my solution to this Frontend Mentor challenge using JavaScript, HTML, and CSS, with a heavy emphasis on Flexbox and Grid properties for a fully responsive design. This project utilizes the services of the shrtcode URL-shortening API, which consumes the fetch requests and returns data in the form of the handy shortened URL link.
